# 弹窗主题（壁纸）方案（定稿）

## 1. 目标

将当前黑底提醒弹窗升级为可配置主题系统，支持：

- 背景（纯色 / 图片）
- 遮罩（颜色、透明度、渐变）
- 文字样式与排版
- 实时预览
- 预设复用与批量应用
- 会员能力预留（free / pro）

---

## 2. 信息架构（双层）

### 2.1 系统设置：主题工坊（重度编辑）

承载完整能力：

- 主题库
- 主题编辑器
- 批量应用

### 2.2 子项新建/编辑：轻量入口

仅承载：

- 主弹窗主题选择
- 休息弹窗主题选择（`splitCount > 1` 时显示）
- 小预览
- “前往主题工坊”按钮

> 不在子项弹窗中放完整颜色/渐变/排版编辑器，避免页面过重。

---

## 3. 数据结构约定

- 主题为独立实体（建议 `popupThemes`）
- 子项仅绑定主题 id
  - `mainPopupThemeId`
  - `restPopupThemeId`

这样可避免样式字段在每条子项重复，便于预设复用和批量应用。

---

## 4. 功能范围

## V1（先做）

- 单图背景 + 纯色背景
- 遮罩基础配置（透明度、颜色）
- 文字基础样式（颜色、大小、对齐、显示顺序）
- 实时预览
- 预设保存/选择
- 批量应用（所有闹钟/所有倒计时，至少二选一）

## V2（后续）

- 文件夹壁纸（随机/顺序）
- 渐变遮罩方向与双端色高级能力
- 更细粒度排版能力（高级位置控制）

---

## 5. 批量应用规则

点击“应用所有”后，弹确认框：

- 作用范围（默认全选）
  - 所有闹钟
  - 所有倒计时
- 应用目标
  - 主弹窗主题
  - 休息弹窗主题

约束：

- 作用范围至少选择一个
- 应用目标至少选择一个
- 显示预计影响条目数量

---

## 6. 会员预留

建议字段：

- `entitlements.popupThemeLevel = 'free' | 'pro'`

建议门控：

- free：纯色/单图/基础文字
- pro：文件夹壁纸、渐变高级参数、排版高级能力

UI 可先显示锁定态提示“会员功能”，后续再接实际鉴权/支付。

---

## 7. 开发顺序（执行清单）

1. 定义 shared 类型与 settings 迁移（主题实体 + 子项绑定）
2. 系统设置主题库 + 主题编辑器基础能力
3. 主进程弹窗渲染接入主题
4. 子项新建/编辑页接入轻量选择入口
5. 批量应用与确认流程
6. 会员门控 UI
